# Selenium-WhatsApp-App-Update-Control
Bu projede **Java** dili ve web tarayıcılarını otomatikleştirmek 
için bir araç paketi olan **Selenium** kullanılmıştır. 
Proje senaryosu olarak üzerinde çalışılmakta olan bir uygulamanın 
yeni sürümünün Play Store ve App Store'da release olup olmadığının 
kontrol  süreci ele alınmıştır. 



Proje çalışma adımları;
* Play Store ve App Store platformlarından hangisinde test işleminin gerçekleştirilmek isteniliğine yönelik seçim yapılması,
* Seçilen platform üzerinde 10 saniye aralıklarla son release tarihi / sürüm numarası kontrolünün yapılaması,
* Yeni sürüm release olduğunda ses ve jOptionPane ile kulalnıcıya bilgi verilmesi,
* Kullanıcı onayı sonrası projenin sonlanmasıdır.
